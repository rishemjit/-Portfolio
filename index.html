About Me

Who Am I?

Hi I'm Rishemjit Kaur working as a Principal Scientist with CSIR-Central Scientific Instruments Organisation Chandigarh. I'm passionate about analysing and interpreting large amount of data from a range of sources using algorithmic, data mining, artificial intelligence, machine learning and statistical tools, in order to make it accessible to food processing instrumentation lab.

My work is at the intersection of deep learning, NLP, and big data analytics. My recent work is on development of neural machine translation systems for low resource languages.

    Other Projects

  • Neural Machine translation for low resource languages
      • Developed a catalog of methods and state-of-art techniques which can be used for translation between languages for different data specifications especially for low resource languages such as Hindi, Tamil, Telugu.
      Provided a set of guidelines to select the possible NMT technique for a given LRL data setting.
  • Data analysis of Kisan Call centres
      • Analyzed millions of Kisan Call Centre queries from 2013 to 2018 for more than 600 districts across India.
      Developed natural language processing models to understand and process the Hinglish queries. Studied the temporal and frequency dynamics of queries using the word embeddings.
        Developed Indian agriculture specific corpus of words for the analysis.
        Studied the spread of crop diseases across different states using the queries.
    • Cross-cultural study of human moral behaviour using social media
        • Led this project along with Dr. Sasahara, Nagoya University and developed strategies for data collection and data interpretation.
        Studied the relationship and roles of different foundations of morality using twitter data.
        Developed singular value decomposition and word2Vec based algorithms for semantic analysis and quantification of moral behaviour to study moral violations.
    • Sentiment Analysis of Tweets for general elections, India, 2014
        • Data collection using appropriate tokens and hashtags related to major political parties and figures.
        Observed and monitored the public sentiments using LIWC dictionary and machine learning algorithms.
    • Odor Space, for Foodpairing, a Belgium based company(November 2021 - April 2022)
    • Bitterness Masking evaluation of Green Tea using machine learning for TATA Consumer Products Ltd., Pune(May 2021 - November 2021)
    • Data logging system for Deg Bhapka process monitoring for FFDC, Kannuaj(September 2017 - October 2017)
    • Identification of volatiles released from rancid cashews and their correlation/characterization, nanoPix ISS (P) Ltd.,Hubli (Jan 2015 - September 2015)
    • Upgradation of Existing Iodine Value Meter, M/S Vaisheshika Systems Ltd., Ambala (Jan 2016 - July 2016)
